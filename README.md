# 🚀 LumiNest — Modern MERN Blog Platform

✨ A sleek, full-stack blogging platform built with the MERN stack — designed with a modern UI, smooth interactions, and real-world functionality.

---


## ✨ Features

🔐 Authentication  
- User registration & login  
- JWT-based authorization  

📝 Blog System  
- Create, edit, delete posts  
- Upload images  
- Protected routes  

🎨 UI/UX  
- Modern glassmorphism design  
- Smooth hover animations  
- Responsive layout  
- Fixed navigation bar  

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- Bootstrap + Custom CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer (image upload)
